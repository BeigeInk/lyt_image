# 神经网络判别龙图

网络是基于AlexNet修改

check里是写好的接口函数

内部调用模型请自行修改

check_ba64是判别base64代码是否为龙图，返回0-1的数值（float类型）

约接近1约可以认为是龙图

因为数据集有限

错误率大概在20%

whole_lt.zip是收集的龙图共三百二十二张

训练需要非龙图的表情包，具体的话请自行收集

建议数量和龙图的数量一样

请自行切分训练集、测试集、验证集
